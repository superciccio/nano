WorkerAtom<P, R> class

An Atom that performs heavy computations in a background isolate.

It listens to a source atom and re-runs the worker task in a background isolate whenever the source changes. The result is exposed as an AsyncState.

Example:

final heavy = WorkerAtom<int, String>(
  source: count,
  worker: (c) => computeSomethingHard(c),
);
