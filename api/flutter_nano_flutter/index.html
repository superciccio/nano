flutter/nano_flutter library

Nano Flutter

A reactive state management and dependency injection library for Flutter. Highly optimized for performance and developer experience.

Classes

AsyncAtomBuilder<T>
A specialized builder for AsyncAtom (or any ValueListenable<AsyncState<T>>).
AtomBuilder<T>
A simplified builder for Atoms. Alias for Watch.
NanoBuildSpy
A utility widget that tracks how many times it has been built.
NanoComponent
A convenience widget that combines a Scope and a reactive view.
NanoConsumer
A widget that tracks Atoms accessed in its builder.
NanoField<T>
A generic builder widget for any input that binds to a FieldAtom<T>.
NanoLayout
Configuration for NanoStack layout.
NanoPage
A standard screen layout that reduces Scaffold boilerplate.
NanoStack
A smart container that applies NanoLayout properties to its children.
NanoStatelessWidget
A substitute for StatelessWidget that automatically tracks Atoms.
NanoTextField
A specialized TextField that binds directly to a FieldAtom<String>.
NanoView<T extends NanoLogic<P>, P>
The Smart View Widget.
Scope
The Dependency Injection Container.
View<T extends NanoLogic<P>, P>
Abstract base class for views that use a NanoLogic.
Watch<T>
Surgical rebuilds.
WatchMany
Batched rebuilds for multiple atoms.
