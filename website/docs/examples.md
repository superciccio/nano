# Examples

Explore these open-source examples to learn how to build apps with Nano.

## [Breaking Bad Quotes](https://github.com/superciccio/nano/tree/main/example/breaking_bad)
A simple app that fetches quotes from an API. Demonstrates:
- `NanoLogic` for state management.
- `NanoView` for UI binding.
- Dependency injection with `Nano.add`.

## [Crypto Tracker](https://github.com/superciccio/nano/tree/main/example/crypto_tracker)
A real-time cryptocurrency tracker. Demonstrates:
- Reactive updates with `Atom`.
- Computed values with `ComputedAtom`.
- List handling.

## [Pokedex](https://github.com/superciccio/nano/tree/main/example/pokedex)
A classic Pokedex app. Demonstrates:
- Complex state management.
- Pagination.
- Custom widgets.

## [Rick and Morty](https://github.com/superciccio/nano/tree/main/example/rick_and_morty)
Character browser for the Rick and Morty API. Demonstrates:
- API integration.
- Infinite scrolling.

## [Sudoku](https://github.com/superciccio/nano/tree/main/example/sudoku)
A Sudoku game. Demonstrates:
- Game logic.
- Undo/Redo history.
- `Atom` updates.

## [Nano Hub](https://github.com/superciccio/nano/tree/main/example/nano_hub)
A hub application showcasing various features.
