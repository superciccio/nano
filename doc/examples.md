# Examples

Here are some examples of how to use Nano in your apps.

## [Breaking Bad Quotes](../example/breaking_bad/)
A simple app that fetches quotes from an API. Demonstrates:
- `NanoLogic` for state management.
- `NanoView` for UI binding.
- Dependency injection with `Nano.add`.

## [Crypto Tracker](../example/crypto_tracker/)
A real-time cryptocurrency tracker. Demonstrates:
- Reactive updates with `Atom`.
- Computed values with `ComputedAtom`.
- List handling.

## [Pokedex](../example/pokedex/)
A classic Pokedex app. Demonstrates:
- Complex state management.
- Pagination.
- Custom widgets.

## [Rick and Morty](../example/rick_and_morty/)
Character browser for the Rick and Morty API. Demonstrates:
- API integration.
- Infinite scrolling.

## [Nano Hub](../example/nano_hub/)
A comprehensive showcase of Nano features.
